Core. LoaderManager

LoaderManager 类用于用于批量加载资源、数据。

批量加载器，单例，可以通过Laya.loader访问。

多线程(默认5个线程)，5个优先级(0最快，4最慢,默认为1) 某个资源加载失败后，会按照最低优先级重试加载(属性retryNum决定重试几次)，如果重试后失败，则调用complete函数，并返回null

构造函数

new LoaderManager()

创建一个新的 LoaderManager 实例。

继承

  • EventDispatcher

成员

maxLoader :int

最大下载线程，默认为5个
Default Value:
  • 5

retryDelay :int

延迟时间多久再进行错误重试，默认立即重试
Default Value:
  • 0

retryNum :int

加载出错后的重试次数，默认重试一次
Default Value:
  • 1
