Core. Matrix

Matrix 类表示一个转换矩阵，它确定如何将点从一个坐标空间映射到另一个坐标空间。

您可以对一个显示对象执行不同的图形转换，方法是设置 Matrix 对象的属性，将该 Matrix 对象应用于 Transform 对象的 Matrix 属性，然后应用该 Transform 对象作为显示对象的 transform 属性。这些转换函数包括平移（x 和 y 重新定位）、旋转、缩放和倾斜。

构造函数

new Matrix(aopt, bopt, copt, dopt, txopt, tyopt)

使用指定参数创建新的 Matrix 对象。
Parameters:
Name Type Attributes Default Description
a Number <optional>
 1 a （可选）缩放或旋转图像时影响像素沿 x 轴定位的值。
b Number <optional>
 0 b （可选）旋转或倾斜图像时影响像素沿 y 轴定位的值。
c Number <optional>
 0 c （可选）旋转或倾斜图像时影响像素沿 x 轴定位的值。
d Number <optional>
 1 d （可选）缩放或旋转图像时影响像素沿 y 轴定位的值。
tx Number <optional>
 0 tx （可选）沿 x 轴平移每个点的距离。
ty Number <optional>
 0 ty （可选）沿 y 轴平移每个点的距离。
