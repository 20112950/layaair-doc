Particle. EmitterBase

EmitterBase 类是粒子发射器类

构造函数

new EmitterBase()

成员

_emissionRate :Number

粒子发射速率
Default Value:
  • 60

_emissionTime :Number

当前剩余发射时间
Default Value:
  • 0

_frameTime :Number

积累的帧时间
Default Value:
  • 0

emissionRate :Number

获取粒子发射速率

minEmissionTime :Number

发射粒子最小时间间隔

particleTemplate :ParticleTemplateBase

advanceTime(passedTimeopt)

Name Type Attributes Default Description
passedTime Number <optional>
 1 passedTime 前进时间

Name Type Description
clearTexture 是否清理贴图数据,若清除贴图数据将无法再播放

Parameters:
Name Type Attributes Default Description

 null.null duration 发射持续的时间(秒)

stop(clearParticles)

停止发射粒子
Parameters:
Name Type Description
