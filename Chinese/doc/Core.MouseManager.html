Core. MouseManager

MouseManager 是鼠标、触摸交互管理器。

鼠标事件流包括捕获阶段、目标阶段、冒泡阶段。 捕获阶段：此阶段引擎会从stage开始递归检测stage及其子对象，直到找到命中的目标对象或者未命中任何对象；
目标阶段：找到命中的目标对象；
*

构造函数

new MouseManager()

成员

(static) enabled :Boolean

是否开启鼠标检测，默认为true

(static) instance :MouseManager

(static) multiTouchEnabled :Boolean

是否开启多点触控

_event :Event

disableMouseEvent :Boolean

是否禁用除 stage 以外的鼠标事件检测。

mouseDownTime :Number

鼠标按下的时间。单位为毫秒。
Default Value:
  • 0

mouseMoveAccuracy :Number

鼠标移动精度。
Default Value:
  • 2

mouseX :Number

canvas 上的鼠标X坐标。
Default Value:
  • 0

mouseY :Number

canvas 上的鼠标Y坐标。
Default Value:
  • 0

方法

runEvent()

执行事件处理。
