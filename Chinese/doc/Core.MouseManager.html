Core. MouseManager

MouseManager 是鼠标、触摸交互管理器。

构造函数

new MouseManager()

成员

(static) enabled :Boolean

是否开启鼠标检测，默认为true

(static) instance :MouseManager

_event :Event

disableMouseEvent :Boolean

是否禁用除 stage 以外的鼠标事件检测。

mouseDownTime :Number

鼠标按下的时间。单位为毫秒。
Default Value:
  • 0

mouseMoveAccuracy :Number

鼠标移动精度。
Default Value:
  • 2

mouseX :Number

canvas 上的鼠标X坐标。
Default Value:
  • 0

mouseY :Number

canvas 上的鼠标Y坐标。
Default Value:
  • 0

方法

runEvent()

执行事件处理。
