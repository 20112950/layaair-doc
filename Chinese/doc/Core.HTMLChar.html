Core. HTMLChar

HTMLChar 是一个 HTML 字符类。

构造函数

new HTMLChar(char, w, h, style)

根据指定的字符、宽高、样式，创建一个 HTMLChar 类的实例。
Parameters:
Name Type Description
char String 字符。
w Number 宽度。
h Number 高度。
style CSSStyle CSS 样式。

成员

char :String

字符。

charNum :Number

字符数量。

height :Number

高度。

isWord :Boolean

表示是否是正常单词(英文|.|数字)。

style :CSSStyle

CSS 样式。

width :Number

宽度。

x :Number

此对象存储的 X 轴坐标值。 当设置此值时，如果此对象有绑定的 Sprite 对象，则改变 Sprite 对象的属性 x 的值。

y :Number

此对象存储的 Y 轴坐标值。 当设置此值时，如果此对象有绑定的 Sprite 对象，则改变 Sprite 对象的属性 y 的值。

方法

获取与此对象绑定的显示对象 Sprite。
Returns:
Type:
Sprite

设置与此对象绑定的显示对象 Sprite 。
Parameters:
Name Type Description
sprite Sprite 显示对象 Sprite 。
