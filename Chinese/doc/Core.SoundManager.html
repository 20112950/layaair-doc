Core. SoundManager

SoundManager 是一个声音管理类。提供了对背景音乐、音效的播放控制方法。

构造函数

new SoundManager()

成员

(static) musicMuted :Boolean

  • 1

声音播放速率。

playMusic(url, loopsopt, completeopt, startTimeopt) → {SoundChannel}

播放背景音乐。背景音乐同时只能播放一个，如果在播放背景音乐时再次调用本方法，会先停止之前的背景音乐，再播发当前的背景音乐。
Parameters:
Name Type Attributes Default Description
url String 声音文件地址。
loops int <optional>
 0 loops 循环次数,0表示无限循环。
complete Handler <optional>
 null complete 声音播放完成回调。
startTime Number <optional>
 0 startTime 声音播放起始时间。
Returns:
Type:
SoundChannel
SoundChannel对象，通过此对象可以对声音进行控制，以及获取声音信息。

playSound(url, loopsopt, completeopt, soundClassopt, startTimeopt) → {SoundChannel}

播放音效。音效可以同时播放多个。
Parameters:
Name Type Attributes Default Description
url String 声音文件地址。
loops int <optional>
 1 loops 循环次数,0表示无限循环。
complete Handler <optional>
 null complete 声音播放完成回调 Handler对象。
soundClass Class <optional>
 null soundClass 使用哪个声音类进行播放，null表示自动选择。
startTime Number <optional>
 0 startTime 声音播放起始时间。
Returns:
Type:
SoundChannel
SoundChannel对象，通过此对象可以对声音进行控制，以及获取声音信息。

setMusicVolume(volume)

设置背景音乐音量。音量范围从 0（静音）至 1（最大音量）。
Parameters:
Name Type Description
volume Number 音量。初始值为1。音量范围从 0（静音）至 1（最大音量）。

setSoundVolume(volume, urlopt)

设置声音音量。根据参数不同，可以分别设置指定声音（背景音乐或音效）音量或者所有音效（不包括背景音乐）音量。
Parameters:
Name Type Attributes Default Description
volume Number 音量。初始值为1。音量范围从 0（静音）至 1（最大音量）。
url String <optional>
 null url (default = null)声音播放地址。默认为null。为空表示设置所有音效（不包括背景音乐）的音量，不为空表示设置指定声音（背景音乐或音效）的音量。

stopAll()

停止播放所有声音（包括背景音乐和音效）。

stopAllSound()

停止播放所有音效（不包括背景音乐）。

stopMusic(url)

停止播放背景音乐（不包括音效）。
Parameters:
Name Type Description
url 声音文件地址。

stopSound(url)

停止声音播放。此方法能够停止任意声音的播放（包括背景音乐和音效），只需传入对应的声音播放地址。
Parameters:
Name Type Description
url String 声音文件地址。
