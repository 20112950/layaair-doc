Core. ResourceVersion

资源版本的生成由layacmd或IDE完成，使用 ResourceVersion 简化使用过程。

调用 enable 启用资源版本管理。

构造函数

new ResourceVersion()

成员

(static) manifest :Object

方法

addVersionPrefix(originURL) → {String}

为加载路径添加版本前缀。
Parameters:
Name Type Description
originURL String 源路径。
Returns:
Type:
String
格式化后的新路径。

enable(manifestFile, callback)

启用资源版本管理。

由于只有发布版本需要资源管理。因此没有资源管理文件时，可以设置manifestFile为null或者不存在的路径。

Parameters:
Name Type Description
manifestFile String 清单（json）文件的路径。
callback Handler 清单（json）文件加载完成后执行。
