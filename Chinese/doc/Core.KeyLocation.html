Core. KeyLocation

KeyLocation 类包含表示在键盘或类似键盘的输入设备上按键位置的常量。

KeyLocation 常数用在键盘事件对象的 keyLocation 属性中。

构造函数

new KeyLocation()

成员

(static) LEFT :uint

Default Value:
  • 1

(static) NUM_PAD :uint

Default Value:
  • 3

(static) RIGHT :uint

Default Value:
  • 2

(static) STANDARD :uint

Default Value:
  • 0
