Core. BitmapFont

BitmapFont 是位图字体类，用于定义位图字体信息。

构造函数

new BitmapFont()

成员

autoScaleSize :Boolean

表示是否根据实际使用的字体大小缩放位图字体大小。

fontSize :Number

当前位图字体字号。
Default Value:
  • 12

letterSpacing :Number

字符间距（以像素为单位）。
Default Value:
  • 0

方法

destroy()

销毁位图字体，调用Text.unregisterBitmapFont 时，默认会销毁。
