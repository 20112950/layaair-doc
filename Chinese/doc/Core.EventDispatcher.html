Core. EventDispatcher

EventDispatcher 类是可调度事件的所有类的基类。

构造函数

new EventDispatcher()

成员

(static) MOUSE_EVENTS :Object

方法

event(type, dataopt) → {Boolean}

派发事件。
Parameters:
Name Type Attributes Default Description
type String 事件类型。
data Object <optional>
 null data 回调数据。 注意：如果是需要传递多个参数 p1,p2,p3,...可以使用数组结构如：[p1,p2,p3,...] ；如果需要回调单个参数 p 是一个数组，则需要使用结构如：[p]，其他的单个参数 p ，可以直接传入参数 p。
Returns:
Type:
Boolean
此事件类型是否有侦听者，如果有侦听者则值为 true，否则值为 false。

hasListener(type) → {Boolean}

检查 EventDispatcher 对象是否为特定事件类型注册了任何侦听器。
Parameters:
Name Type Description
type String 事件的类型。
Returns:
Type:
Boolean
如果指定类型的侦听器已注册，则值为 true；否则，值为 false。

Parameters:

Returns:
Type:
Boolean
