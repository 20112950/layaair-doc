Core. TextStyle

文本的样式类

构造函数

new TextStyle()

继承

  • SpriteStyle

成员

(static) EMPTY :TextStyle

align :String

表示使用此文本格式的文本段落的水平对齐方式。

Default Value:
  • "left"

asPassword :Boolean

指定文本字段是否是密码文本字段。

如果此属性的值为 true，则文本字段被视为密码文本字段，并使用星号而不是实际字符来隐藏输入的字符。如果为 false，则不会将文本字段视为密码文本字段。

bgColor :String

文本背景颜色，以字符串表示。

bold :Boolean

是否为粗体

borderColor :String

文本边框背景颜色，以字符串表示。

currBitmapFont :BitmapFont

当前使用的位置字体。

italic :Boolean

表示使用此文本格式的文本是否为斜体。
Default Value:
  • false

leading :Number

垂直行间距（以像素为单位）

overflow :String

溢出处理

padding :Array

默认边距信息

[左边距，上边距，右边距，下边距]（边距以像素为单位）

stroke :Number

描边宽度（以像素为单位）。

默认值0，表示不描边。
Default Value:
  • 0

strokeColor :String

描边颜色，以字符串表示。

Default Value:
  • "#000000";

underline :Boolean

是否显示下划线

underlineColor :String

下划线颜色

valign :String

表示使用此文本格式的文本段落的垂直对齐方式。

Default Value:
  • "top"

wordWrap :Boolean

表示使用此文本格式的文本字段是否自动换行。

如果 wordWrap 的值为 true，则该文本字段自动换行；如果值为 false，则该文本字段不自动换行。
Default Value:
  • false。
