parseInt(str, radixopt) → {int}

解析一个字符串，并返回一个整数。和JS原生的parseInt不同：如果str为空或者非数字，原生返回NaN，这里返回0
Parameters:
Name Type Attributes Default Description
str String 要被解析的字符串
radix int <optional>
 0 radix 表示要解析的数字的基数。该值介于 2 ~ 36 之间。如果它以 “0x” 或 “0X” 开头，将以 16 为基数。如果该参数小于 2 或者大于 36，则 parseInt() 将返回 NaN。
Returns:
Type:
int
返回解析后的数字
