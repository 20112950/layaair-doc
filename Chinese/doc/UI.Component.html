UI. Component

Component 是ui控件类的基类。

生命周期：preinitialize > createChildren > initialize > 组件构造函数

构造函数

new Component()

创建一个新的 Component 实例。

继承

  • Sprite

成员

anchorX :Number

X轴锚点，值为0-1

anchorY :Number

Y轴锚点，值为0-1

bottom :Number

从组件底边到其内容区域底边之间的垂直距离（以像素为单位）。

centerX :Number

在父容器中，此对象的水平方向中轴线与父容器的水平方向中心线的距离（以像素为单位）。

centerY :Number

在父容器中，此对象的垂直方向中轴线与父容器的垂直方向中心线的距离（以像素为单位）。

comXml :Object

XML 数据。

dataSource :Object

数据赋值，通过对UI赋值来控制UI显示逻辑。

简单赋值会更改组件的默认属性，使用大括号可以指定组件的任意属性进行赋值。

示例
以下示例中， <code>label1、checkbox1</code> 分别为示例的name属性值。
		   <listing version="3.0">
		   //默认属性赋值
		   dataSource = {label1: "改变了label", checkbox1: true};//(更改了label1的text属性值，更改checkbox1的selected属性)。
		   //任意属性赋值
		   dataSource = {label2: {text:"改变了label",size:14}, checkbox2: {selected:true,x:10}};
		</listing>

disabled :Boolean

是否禁用页面，设置为true后，会变灰并且禁用鼠标。

displayHeight :Number

对象的显示高度（以像素为单位）。

displayWidth :Number

对象的显示宽度（以像素为单位）。

gray :Boolean

是否变灰。

height :Number

表示显示对象的高度，以像素为单位。

注：当值为0时，高度为自适应大小。

left :Number

从组件左边到其内容区域左边之间的水平距离（以像素为单位）。

measureHeight :Number

显示对象的实际显示区域高度（以像素为单位）。

measureWidth :Number

显示对象的实际显示区域宽度（以像素为单位）。

right :Number

从组件右边到其内容区域右边之间的水平距离（以像素为单位）。

scaleX :Number

scaleY :Number

tag :Object

对象的标签。

toolTip :Object

鼠标悬停提示。

可以赋值为文本 String 或函数 Handler ，用来实现自定义样式的鼠标提示和参数携带等。

示例
以下例子展示了三种鼠标提示：
		   <listing version="3.0">
		   private var _testTips:TestTipsUI = new TestTipsUI();
		   private function testTips():void {
		   //简单鼠标提示
		   btn2.toolTip = "这里是鼠标提示&lt;b&gt;粗体&lt;/b&gt;&lt;br&gt;换行";
		   //自定义的鼠标提示
		   btn1.toolTip = showTips1;
		   //带参数的自定义鼠标提示
		   clip.toolTip = new Handler(this,showTips2, ["clip"]);
		   }
		   private function showTips1():void {
		   _testTips.label.text = "这里是按钮[" + btn1.label + "]";
		   tip.addChild(_testTips);
		   }
		   private function showTips2(name:String):void {
		   _testTips.label.text = "这里是" + name;
		   tip.addChild(_testTips);
		   }
		</listing>

top :Number

从组件顶边到其内容区域顶边之间的垂直距离（以像素为单位）。

width :Number

表示显示对象的宽度，以像素为单位。

注：当值为0时，宽度为自适应大小。

方法

callLater(method, argsopt)

延迟运行指定的函数。

在控件被显示在屏幕之前调用，一般用于延迟计算数据。

See:
  • Component.html#runCallLater()">#runCallLater()

Name Type Attributes Default Description
method function 要执行的函数的名称。例如，functionName。
args Array <optional>
 null args 传递给 method 函数的可选参数列表。

changeSize()

重新调整对象的大小。

commitMeasure()

立即执行影响宽高度量的延迟调用函数。

See:
  • Component.html#callLater()">#callLater()

destroy()
