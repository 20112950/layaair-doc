Core. Tween

Tween 是一个缓动类。使用实现目标对象属性的渐变。

构造函数

new Tween()

成员

progress :Number

update :Handler

更新回调，缓动数值发生变化时，回调变化的值

方法

clear()

停止并清理当前缓动。

clear(tween)

清理某个缓动。
Parameters:
Name Type Description
tween Tween 缓动对象。

clearAll(target)

清理指定目标对象上的所有缓动。
Parameters:
Name Type Description
target Object 目标对象。

complete()

立即结束缓动并到终点。

from(target, props, duration, easeopt, completeopt, delayopt, coverBefore, autoRecover) → {Tween}

从props属性，缓动到当前状态。
Parameters:
Name Type Attributes Default Description
target Object 目标对象(即将更改属性值的对象)。
props Object 变化的属性列表，比如{x:100,y:20,ease:Ease.backOut,complete:Handler.create(this,onComplete),update:new Handler(this,onComplete)}。
duration int 花费的时间，单位毫秒。
ease function <optional>
 null ease 缓动类型，默认为匀速运动。
complete Handler <optional>
 null complete 结束回调函数。
delay int <optional>
 0 delay 延迟执行时间。
coverBefore Boolean 是否覆盖之前的缓动。
autoRecover Boolean 是否自动回收，默认为true，缓动结束之后自动回收到对象池。
Returns:
Type:
Tween
返回Tween对象。

pause()

暂停缓动，可以通过resume或restart重新开始。

recover()

回收到对象池。

restart()

重新开始暂停的缓动。

resume()

恢复暂停的缓动。

setStartTime(startTime)

设置开始时间。
Parameters:
Name Type Description
startTime Number 开始时间。
