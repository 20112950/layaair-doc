Core. URL

URL 类用于定义地址信息。

构造函数

new URL()

创建一个新的 URL 实例。

成员

自定义url格式化。例如： customFormat = function(url:String):String{}

(static) rootPath :String

根路径。
Default Value:
  • ""

(static) version :Object

版本号。
