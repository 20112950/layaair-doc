Animation. MovieClip

MovieClip 用于播放经过工具处理后的 swf 动画。

构造函数

new MovieClip(parentMovieClipopt)

创建一个 MovieClip 实例。
Parameters:
Name Type Attributes Default Description
parentMovieClip MovieClip <optional>
 null parentMovieClip 父MovieClip,自己创建时不需要传该参数

继承

  • Sprite

成员

basePath :String

资源根目录。

count :int

帧总数。

index :int

当前播放索引。

interval :int

播放间隔(单位：毫秒)。
Default Value:
  • 30

loop :Boolean

是否循环播放

playing :Boolean

是否在播放中

url :String

方法

addLabel(label, index)

增加一个标签到index帧上，播放到此index后会派发label事件
Parameters:
Name Type Description
label String 标签名称
index int 索引位置

destroy(destroyChild)

销毁此对象。以及销毁引用的Texture

Parameters:
Name Type Description
destroyChild Boolean 是否同时销毁子节点，若值为true,则销毁子节点，否则不销毁子节点。

gotoAndStop(frame)

跳到某帧并停止播放动画。
Parameters:
Name Type Description
frame int 要跳到的帧

load(url, atlas, atlasPathopt)

加载资源。
Parameters:
Name Type Attributes Default Description
url String swf 资源地址。
atlas Boolean 是否使用图集资源
atlasPath String <optional>
 null atlasPath 图集路径，默认使用与swf同名的图集

play(indexopt)

播放动画。
Parameters:
Name Type Attributes Default Description
index int <optional>
 0 index 帧索引。

playTo(start, end, completeopt)

从开始索引播放到结束索引，结束之后出发complete回调
Parameters:
Name Type Attributes Default Description
start int 开始索引
end int 结束索引
complete Handler <optional>
 null complete 结束回调

removeLabel(label)

删除某个标签
Parameters:
Name Type Description
label String 标签名字，如果label为空，则删除所有Label

stop()

停止播放动画。
