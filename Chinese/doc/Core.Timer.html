Core. Timer

Timer 是时钟管理类。它是一个单例，可以通过 Laya.timer 访问。

构造函数

new Timer()

创建 Timer 类的一个实例。

成员

currFrame :int

当前的帧数。
Default Value:
  • 0

currTimer :Number

当前时间。
Default Value:
  • null.now

delta :int

*两帧之间的时间间隔,单位毫秒。

scale :Number

时针缩放。
Default Value:
  • 1
