Core. ResourceManager

ResourceManager 是资源管理类。它用于资源的载入、获取、销毁。

构造函数

new ResourceManager()

创建一个 ResourceManager 实例。

成员

(static) currentResourceManager :ResourceManager

当前资源管理器。

(static) sortedResourceManagersByName :null

排序后的资源管理器列表。

(static) systemResourceManager :ResourceManager

系统资源管理器。

autoRelease :Boolean

是否启用自动释放机制。

autoReleaseMaxSize :int

自动释放机制的内存触发上限,以字节为单位。

id :int

唯一标识 ID 。

memorySize :int

此管理器所管理资源的累计内存，以字节为单位。

name :String

名字。
