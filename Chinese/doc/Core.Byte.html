Core. Byte

Byte 类提供用于优化读取、写入以及处理二进制数据的方法和属性。

构造函数

new Byte(dataopt)

创建一个 Byte 类的实例。
Parameters:
Name Type Attributes Default Description
data Object <optional>
 null data 用于指定元素的数目、类型化数组、ArrayBuffer。

成员

(static) BIG_ENDIAN :String

Default Value:
  • "bigEndian"

(static) LITTLE_ENDIAN :String

Default Value:
  • "littleEndian"

buffer :ArrayBuffer

获取此对象的 ArrayBuffer数据,数据只包含有效数据部分 。

bytesAvailable :int

可从字节流的当前位置到末尾读取的数据的字节数。

endian :String

字节顺序。

length :int

字节长度。

pos :int

当前读取到的位置。
