UI. LayoutStyle

LayoutStyle 是一个布局样式类。

构造函数

new LayoutStyle()

成员

(static) EMPTY :LayoutStyle

anchorX :Number

X锚点，值为0-1。
Default Value:
  • NaN

anchorY :Number

Y锚点，值为0-1。
Default Value:
  • NaN

bottom :Number

表示距底边的距离（以像素为单位）。
Default Value:
  • NaN

centerX :Number

表示距水平方向中心轴的距离（以像素为单位）。
Default Value:
  • NaN

centerY :Number

表示距垂直方向中心轴的距离（以像素为单位）。
Default Value:
  • NaN

enable :Boolean

一个布尔值，表示是否有效。

left :Number

表示距左边的距离（以像素为单位）。
Default Value:
  • NaN

right :Number

表示距右边的距离（以像素为单位）。
Default Value:
  • NaN

top :Number

表示距顶边的距离（以像素为单位）。
Default Value:
  • NaN
