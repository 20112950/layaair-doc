Core. Text

Text Permettant de créer des objets d'affichage de type à affichage de texte.

Constructor

new Text()

Créer un nouveau Text Des exemples.
示例
Le code suivant, crée un <code>Text</code> Des exemples. 
<listing version="3.0">
package
{
import laya.display.Text;
public class Text_Example
{
public function Text_Example()
{
Laya.init(640, 800);// Le mode de réglage de largeur élevé, jeu de rendre la toile. 
Laya.stage.bgColor = "#efefef";// Couleur de fond de la toile. 
onInit();
}
private function onInit():void
{
var text:Text = new Text();// Créer un  Text  Un objet d'instance de la classe  Text  . 
text.text = "这个是一个 Text 文本示例。";
text.color = "#008fff";// Ensemble  Text  Couleur du texte. 
text.font = "Arial";// Ensemble  Text  Police du texte. 
text.bold = true;// Ensemble  Text  Affichage de texte en gras. 
text.fontSize = 30;// Ensemble  Text  La taille de la police. 
text.wordWrap = true;// Ensemble  Text  Du renvoi à la ligne automatique. 
text.x = 100;// Ensemble  Text  La valeur X de propriété d'objet, des moyens de commande  Text  La position d'affichage de l'objet. 
text.y = 100;// Ensemble  Text  Les valeurs y des attributs d'objet, des moyens de commande  Text  La position d'affichage de l'objet. 
text.width = 300;// Ensemble  Text  La largeur. 
text.height = 200;// Ensemble  Text  En hauteur. 
text.italic = true;// Ensemble  Text  Afficher le texte en italique. 
text.borderColor = "#fff000";// Ensemble  Text  Couleur du texte à la frontière. 
Laya.stage.addChild(text);// Va  Text  Ajouter à la liste d'affichage. 
}
}
}
