Core. Graphics

Graphics Classe est utilisée pour créer un dessin d'objet d'affichage.Graphique de rendu peut être simultanément une pluralité de bitmaps ou diagramme vectoriel, peut également être combiné avec Save, Restore, transformation, scale, rotate, translate, alpha, tels que les changements de l'instruction de dessin. Graphique de flux de commande de moyens de stockage afin, par cmds à accéder à des attributs de tous les flux de commande.L'objet graphique est plus léger que Sprite, de l'utilisation rationnelle de l'énergie améliore la performance d'application (par exemple, le nombre de noeuds de dessin modifiée pour un ensemble de noeuds de commande graphique, ce qui permet de réduire la consommation de grandes quantités de création de noeud).
