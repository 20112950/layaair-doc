Core. KeyBoardManager

KeyBoardManager Est une classe de gestion d'événements de clavier.

Le type dans le navigateur reçoit des événements de clavier, et à distribuer l'événement. Un événement si Stage.focus est vide uniquement à partir de l'étage de distribuer l'événement, ou à partir de Stage.focus objet de bulles de l'événement a été distribué. *

L'utilisateur peut, par l'intermédiaire de moyens de Code Laya.stage.focus = someNode pour régler le focus de l'objet.

L'utilisateur peut e.keyCode uniforme selon l'objet d'événement afin de déterminer le type de touche dans l'attribut de navigateur compatible avec différents.
