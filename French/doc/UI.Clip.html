UI. Clip

Clip La classe est la table de bits de tranchage de l'animation.

Clip Une photo, selon le nombre de segmentation horizontale clipX Segmentation verticale, la quantité de clipY La largeur de la ou chaque tranche de segmentation horizontale clipWidth , la hauteur de la segmentation de chaque tranche verticale clipHeight , *

L'image d'ensemble et de deux composants: le support chargé de manière asynchrone, par exemple clip.skin = "ABC / xxx.png", d'autres composantes ui ne sont pas de support de charge asynchrone.

<listing version="3.0">
import Clip = laya.ui.Clip;
import Handler = laya.utils.Handler;
class Clip_Example {
    private clip: Clip;
    constructor() {
        Laya.init(640, 800);// Ensemble de jeu de la toile de large. 
        Laya.stage.bgColor = "#efefef";// Couleur de fond de la toile. 
        this.onInit();
    }
    private onInit(): void {
        this.clip = new Clip("resource/ui/clip_num.png", 10, 1);// Créer un  Clip  Un objet d'instance de la classe  Clip  Le nombre et la peau, entrant la segmentation horizontale et verticale de quantité de division. 
        this.clip.autoPlay = true;// Ensemble  Clip  Une animation de jeu automatique. 
        this.clip.interval = 100;// Ensemble  Clip  La lecture d'animation de l'intervalle de temps. 
        this.clip.x = 100;// Ensemble  Clip  La valeur X de propriété d'objet, des moyens de commande  Clip  La position d'affichage de l'objet. 
        this.clip.y = 100;// Ensemble  Clip  Les valeurs y des attributs d'objet, des moyens de commande  Clip  La position d'affichage de l'objet. 
        this.clip.on(laya.events.Event.CLICK, this, this.onClick);// Donner  Clip  Cliquez sur Ajouter la fonction d'événement d'interception. 
        Laya.stage.addChild(this.clip);// Ce  Clip  L'objet est ajouté à la liste d'affichage. 
    }
    private onClick(): void {
        console.log("clip 的点击事件侦听处理函数。clip.total=" + this.clip.total);
        if (this.clip.isPlaying == true) {
            this.clip.stop();// Arrêter l'animation. 
        } else {
            this.clip.play();// De l'animation. 
        }
    }
}
